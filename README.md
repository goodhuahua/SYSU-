# SYSU BBS

## 1. 环境要求

|         | 版本查看语句      | 本项目使用的版本 |
| ------- | ----------------- | ---------------- |
| Go      | `go version`      | 1.17.6           |
| Node.js | `node --version`  | 16.14.0          |
| Vue.js  | `vue --version`   | 3.2.37           |
| MySQL   | `mysql --version` | 8.0.28           |

## 2. 数据库准备

需要自己建立 category 表与 question 表

## 3. 启动项目

从终端进入 blog_server，输入以下语句启动后端：

```
go run main.go
```

从终端进入 blog_client，输入以下语句启动前端：

```
yarn

yarn dev

```
